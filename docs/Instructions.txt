We have developed two processors named E-ntel Processor Pro and Base E-ntel Processor. The Base Processor is designed to perform only the task specified in the lab. In both versions, the value in register R7 is displayed in 2’s complement format using the leftmost digit of the seven-segment display and the LEDs U16 to V19. The Jump, Zero, and Overflow flags are indicated by LEDs P1, N3, and L1, respectively. The Pro version includes creative enhancements: the leftmost digit shows only the modulus (absolute value) of the selected value, while the LEDs U16 to V19 still show the full value in 2’s complement. A switch at V17 allows toggling the display between the final output (all-out) and register R7. The second digit of the display shows a minus sign if the selected value is negative. The third digit shows the current memory address (0–7) from ROM, and the rightmost digit shows the next memory address. A dedicated reset button is located at U18 and is used to initialize the processor in both versions.
